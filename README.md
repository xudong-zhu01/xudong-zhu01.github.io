# xudong-zhu01.github.io

Personal academic website for Xudong Zhu, published with GitHub Pages and Jekyll.

## Structure

- Root HTML files contain the main profile, CV, publications, teaching, and blog pages.
- `_includes/` contains shared head, navigation, and footer markup.
- `_layouts/post.html` renders Markdown posts from `_posts/`.
- `assets/` contains optimized website images.
- `documents/` contains the downloadable CV.

## Maintenance

- Update the CV PDF and the visible last-updated label together.
- Add new writing as `_posts/YYYY-MM-DD-slug.md`.
- Keep page-specific titles and descriptions concise for search and social previews.
- Restore or remove the unavailable historical article images referenced by older posts.
