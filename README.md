## About This Site

This is my personal academic website, showcasing my research in international history, statecraft, and contemporary governance. The site features:

- **Publications & Research**: Comprehensive list of academic papers, conference presentations, and ongoing research
- **Professional Resume**: Detailed CV with academic credentials and experience
- **Blog**: Insights on international relations, history, and policy analysis
- **Modern Design**: Clean, professional interface optimized for academic presentation

## Technical Stack

- **Framework**: Jekyll static site generator
- **Theme**: [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) - A minimal, responsive, and feature-rich Jekyll theme
- **Hosting**: GitHub Pages
- **Analytics**: Google Analytics
- **Fonts**: Misans Font for enhanced typography

## Local Development

```bash
# Install dependencies
bundle install

# Run local server
bundle exec jekyll serve

# Visit http://localhost:4000
```
